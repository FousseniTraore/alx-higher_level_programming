"""
This module defines a function that adds 2 integers a and b.
>>> add_integer(2, 3)
5
"""
""" The function return an integer when you pass two integer as parameters."""
::
    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(1, 2)
    3
::
    >>> add_integer(2, -4)
    -2

""" The function return an integer when you pass two floating numbers as paramters"""
::
    >>> add_integer(1.2, 2.6)
    4

""" The function return an integer when you pass an integer and a floating number."""
::
    >>> add_integer(1, 2.3)
    3

""" The function works when you pass only the first parameter"""
::
    >>> add_integer(4)
    102

""" The function does not work with two non-integer number the function return an error message.""""
>>> add_integer("hello", 3)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2, "hello")
    Traceback (most recent call last):
    TypeError: b must be an integer

::

    >>> add_integer(None)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2.3, None)
    Traceback (most recent call last):
    TypeError: b must be an integer

If both arguments are non-integers and non-floats, a TypeError message is only
printed for the first argument.

::

    >>> add_integer("hello", "there")
    Traceback (most recent call last):
    TypeError: a must be an integer

The function will fail if infinity is provided.

::

    >>> add_integer(float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot convert float infinity to integer
     

::

    >>> add_integer(2, float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot convert float infinity to integer

And again with NaN numbers.

::

    >>> add_integer(float('nan'))
    Traceback (most recent call last):
    ValueError: cannot convert float NaN to integer

::

    >>> add_integer(2, float('nan'))
    Traceback (most recent call last):
    ValueError: cannot convert float NaN to integer
